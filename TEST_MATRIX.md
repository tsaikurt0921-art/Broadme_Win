# Broadme Win10/11 Test Matrix

## 1. 基本啟動流程
- 啟動 -> Launch 1.5 秒 -> 序號檢查
- 有有效序號：直接進主控頁
- 無序號/失效：進綁定頁，可成功綁定後進主控頁

## 2. 串流可用性
- 選擇解析度/FPS/網卡後開始廣播
- 主機本機可開 `http://<ip>:8080/stream`
- 同網段手機可開 `http://<ip>:8080/stream`
- 結束廣播後連線中斷

## 3. 控制授權
- 平板開 `http://<ip>:8080/control`
- 控制端顯示 PIN
- 主機輸入 PIN 授權
- 非授權 PIN 端應持續等待或被拒
- 撤銷授權後，控制端需重新授權

## 4. 遠端控制操作
- 滑鼠移動/單擊/雙擊
- 滾輪垂直與水平
- annotationStart/move/end/clearAnnotations

## 5. 上傳照片
- 控制端上傳圖片
- 主機可收到並以預設程式開啟

## 6. 多網卡與邊界
- Wi-Fi + Ethernet 同時存在時切換網卡測試
- 網卡無效 IP（169.254.*）不可作為主推廣播 URL
- 停止廣播後再次開始，URL 與網卡選擇一致

## 7. Windows 版本覆蓋
- Windows 10 22H2
- Windows 11 23H2 / 24H2
- 100% / 125% / 150% DPI
- 單螢幕 / 雙螢幕

## 8. 打包後驗證
- 安裝後首次啟動防火牆提示
- 不用管理員權限可正常執行主流程
- `wwwroot/control.html` 與資產存在於輸出目錄
